About This Project

AI情報ブログとは

AI情報ブログは、注目しているYouTubeチャンネルからキーワードを拾い上げ、AIによる再調査と要約を経て記事化する自動化ワークフローの実験場です。

MVP段階では GitHub Pages + GitHub Actions のみで構成し、外部サービスへの依存を最小限に抑えています。

主な機能

  • 監視対象は data/sources.json で管理
  • 朝（JST 09:00）と夜（JST 21:00）に自動的にドラフト生成フローを実行
  • 生成されたドラフトは posts/generated-drafts/ に保存し、人手で公開可否を判断
  • Google Custom Search APIでトピックの関連情報を収集
  • OpenAI GPT-4を使用して記事を自動生成

技術スタック

  • ホスティング: GitHub Pages
  • 自動化: GitHub Actions (cron スケジュール)
  • データソース: YouTube RSS Feed
  • AI生成: OpenAI GPT-4o-mini
  • 検索API: Google Custom Search
  • フロントエンド: Vanilla JavaScript + CSS3

今後の展望

自動生成プロセスへのフィードバックループを加え、より信頼できる技術ブログとして拡張していきます。具体的には以下の機能を検討中です:

  • 記事品質の自動評価システム
  • ユーザーフィードバックの収集と反映
  • マルチソース対応（Twitter、Reddit等）
  • カテゴリ別のフィルタリング機能
